Nintendo Quiz App

Description

This is a small web app I created as a school project. I took a great deal of pride in going the extra mile anywhere I could without spreading myself too thin, while still meeting the requirements for the project. A lot of my interest and character went into generating the final project you see. To build this app I utilized the best of HTML5 to hardcode the landing page. It was then styled with CSS3 after I built the necessary JavaScript functions to handle interactivity. I also made use of jQuery methods and event listeners to keep the code clean and concise.
